Boolean World

System based on Trues and Falses

The basic unit of any computer data is binary digits, 0 or 1. Although every data is stored with tons of binary digits, programmers don't work with them that much nowadays. What if we rebuild a system based on binary values?

In Python, boolean values, Trues and Falses, are more like binary digits. The challenge is to build common data types with only given boolean class and array class from the module boolean.
